Private seller analytics for the Reveriee Etsy shop

This internal tool connects to the official Etsy Open API v3 via OAuth to fetch daily, minimal shop metrics. Data is used to build non‑public dashboards for operations and inventory planning.

  • Read‑only
  • Minimum scopes
  • No scraping
  • No resale

What it does

Aggregates listing‑level and variation‑level stats such as impressions, favorites, orders, unit price, and shop sections into a daily snapshot for analysis.

How data is handled

Only authorized shop data is accessed. Storage follows the principle of data minimization and retention aligned to operational needs. You can request deletion at any time.

Security

API credentials are stored securely. In the unlikely event of a suspected data incident we will notify the affected seller and Etsy within 24 hours.
